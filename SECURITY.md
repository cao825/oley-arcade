# Security Policy

## Reporting a Vulnerability

If you find a security issue in oleyarcade.com or this repository,
please report it privately rather than opening a public issue.

**Contact:** CraigOley@gmail.com

Please include:
- A description of the issue
- Steps to reproduce it
- The page or URL where you found it

We'll do our best to respond within a few days. This is a personal
hobby project, so please be patient.

## Scope

This site is a static collection of browser games hosted on Vercel.
It does not collect personal data, use accounts, or process payments.

Thanks for helping keep the site safe.
